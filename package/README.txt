Bulk Product Photo Renamer for Linux

What to click:
- Launch_Bulk_Product_Photo_Renamer

What it does:
- Renames/copies product photos into clean numbered filenames
- Uses the keyword or product name you enter
- Helps organize product photo filenames

What it does NOT do:
- Does not enhance photo quality
- Does not compress images
- Does not resize images
- Does not remove backgrounds
- Does not color-correct photos
- Does not edit image pixels

How to run:
1) Right-click Launch_Bulk_Product_Photo_Renamer
2) Click Run as Program

If it doesn't open:
- Install prerequisites:
  sudo apt update && sudo apt install python3 python3-tk python3-venv

Troubleshooting log:
- /tmp/product_photo_renamer_launcher.log
